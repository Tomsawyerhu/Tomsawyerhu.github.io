## IDEA Plugin
IDEA通过插件来扩展、增强IDE的功能，插件主要分为5种类型：
* UI Themes
* Custom language support
* Framework integration
* Tool integration
* User interface add-ons

详情请参考[types-of-plugins](https://plugins.jetbrains.com/docs/intellij/types-of-plugins.html).
